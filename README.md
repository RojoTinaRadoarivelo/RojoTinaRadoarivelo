## 🚀 Front-end Architecture

![React Query](https://img.shields.io/badge/React_Query-FF4154?style=for-the-badge&logo=reactquery&logoColor=white)
![RxJS](https://img.shields.io/badge/RxJS-B7178C?style=for-the-badge&logo=rxjs&logoColor=white)
![Zod](https://img.shields.io/badge/Zod-3E67B1?style=for-the-badge)

- **Architecture front moderne et modulaire** : hooks, services, context, modules Angular
- **Gestion avancée du state serveur** : React Query (cache, invalidation, mutations), RxJS (observables, reactive state)
- **Validation de formulaires** : Zod + React Hook Form / Angular Reactive Forms + Validators dynamiques
- **Appels API sécurisés** : abstractions type `safeApiCall`, services centralisés, gestion uniforme des erreurs
- **Internationalisation** : i18next (React) & Transloco (Angular)
- **UI dynamique et responsive** : dark mode, animations fluides, responsive design
